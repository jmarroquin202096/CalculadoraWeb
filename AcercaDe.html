Como se Realizo la Calculadora

Para la Siguiente Calculadora, se realizo en HTML 5, CSS y JavaScript. Podemos ver que se realizo un Menu, donde de encuentra las 3 opciones de Inicio, Calculadora y Acerca de.

El Menu se realizo de una manera muy creativa y viendo un video:

Link Menu

Para la calculadora se realizo de una manera muy creativa y viendo un video aplicando un estilo unico del desarrollador:

Link Calculadora

En el Menu podemos visulazar un estilo de CSS unico ya que se uso una pagina web para descargar los iconos, donde se proporcionar el codigo y se puede unir al menu para darle un toque unico. Para la calculadora se le implementaron 2 opciones mas la de Porcentaje y la de Masmenos, y se le agrego un estilo de letra google y tambien se realizo la importacion requerida.
